# Ivy official node sdk

### Getting started

Installation:

```
npm i https://github.com/MOF-IVY/ivy-node-sdk
```

Usage:

```ts
import { IvySDK } from 'ivy-node-sdk';

const sdk = newIvySDK();

sdk.log('It works!');
```
